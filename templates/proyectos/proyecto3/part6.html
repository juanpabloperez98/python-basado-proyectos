Parte 6

Ya se han realizado las funciones de crear libros y de crear préstamos, ahora viene la parte de eliminar un libro de la lista de libros, esta función es la encargada de eliminar el libro especificado por el usuario de la lista de libros, la función recibe como parámetro la lista de libros,y retorna la misma lista de libros pero actualizada, es decir sin el libro que se acabo de eliminar.

¿Como se realizara?

Al igual que las funciones anteriores, esta se empieza pidiendo el isbn del libro a eliminar, después se declara una variable llamada índice, esta servirá para determinar cuál es el índice de la lista en la que está ubicado el libro a eliminar, luego con un ciclo for, se recorre la lista de libros por índice y se valida si el libro que está en la posición i, con el valor ‘isbn’ es igual al isbn que ingresó el usuario, de ser así la variable que se llamó índice, se iguala al índice en el que se encuentra el libro. Luego se validará que el índice sea diferente de -1, si esto se cumple significa que si hay un libro para eliminar, entonces se elimina de la lista el elemento el libro en la posición índice, si no se cumple la condición significa que no se encontro ningun libro con ese isbn y se imprime que no hay libro para eliminar, por último se retorna la lista de libros (actualizada dado el caso de que si se haya podido eliminar el libro).

Solución

              
      1def eliminar_libros(libros):
      2    isbn = int(input("Ingrese ISBN: "))
      3    indice = -1
      4    for i in range(len(libros)):
      5        if(libros[i]['isbn'] == isbn):indice = i
      6    if (indice != -1):
      7        libros.pop(indice)
      8        print("Libro eliminado con exitp")
      9    else:print("Libro no encontrado intente de nuevo")
      10  return libros

Linea # 7

Si la condición anterior se cumple se debe eliminar de la lista de libros el libro encontrado según la variable indice ¿Como se elimina de una lista?

Felicitaciones

Si indice es diferente de -1, es porque su valor corresponde al índice en la lista en el que se encuentra el libro a eliminar, se le pasa como parámetro al método pop().
