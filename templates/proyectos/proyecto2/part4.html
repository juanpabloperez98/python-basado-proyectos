Parte 4

En este cuarto apartado se creará la función crear_jugador, esta función se encarga de crear un jugador nuevo cada vez que el usuario escoja la opción 2. Esta función recibe como parámetros la lista de jugadores y devuelve la misma lista de jugadores pero actualizada (es decir, con el nuevo jugador registrado), en este apartado se tocará el tema de diccionarios, así que si usted no ha visto la parte teórica de este tema o siente que tiene falencias sobre este le recomendamos que mire nuevamente la teoría ya que es de suma importancia que entienda sobre cómo se trabajan con diccionarios en python.

¿Como se realizara?

En esta función lo primero que se va hacer es pedirle al usuario que ingrese el nombre del jugador que desea registrar. Después se debe validar que ningún jugador esté registrado con el nombre que acabo de ingresar el usuario, para hacer esto se debe entender que cada elemento de la lista de jugadores corresponde a un diccionario que tiene como claves (‘nombre’,’partidas_ganadas’), entonces para recorrer esta lista se debe hacer un ciclo for, el cual va iterar entre cada uno de los elementos de la lista, de esta manera en cada iteración lo que se va a obtener es un diccionario que tiene como claves (‘nombre’,’partidas_ganadas’). Entendiendo esto lo que sigue ahora es validar si lo que hay en ‘nombre’ es igual a el nombre ingresado por el usuario, de ser así se va a retornar la lista de jugadores tal y como se tenía antes, (es decir sin ningún cambio). Si después de iterar toda la lista no se encuentra ninguna coincidencia de nombres entre los elementos de la lista jugadores y el nombre ingresado por el usuario, entonces significa que se puede crear un jugador con el nombre escogido por el usuario. Para hacer esto se va a crear un diccionario con las claves ‘nombre’ y ‘partidas_ganadas’ y se guarda en una variable llamada jugador, después se iguala jugador en la clave nombre a el nombre que el usuario ingresó, por defecto el valor de partidas_ganadas es igual a 0. Luego se agrega a la lista de jugadores el diccionario creado (o sea el que está en la variable jugador), y se retorna la lista jugadores actualizada(es decir, con el nuevo jugador agregado).
