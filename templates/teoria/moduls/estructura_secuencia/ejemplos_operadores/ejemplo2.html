Ejemplo 2

Enunciado

Se tiene una variable llamada base que esta inicializada en 2, se necesita realizar un programa que le pida al usuario que ingrese un numero, el cual representara el exponente al que se quiere elevar la variable base. Imprimir el valor del numero elevado.

Solución

        
1base = 2
2expo = int(input("Ingrese exponente: "))
3base**=expo
4print("El numero elevado al exponente es: ",base)
Salida
Reiniciar

Explicación

        
1Declaramos la variable "base" y la igualamos a 2
2Le pedimos al usuario que ingrese el exponente y lo guardamos en la variable "expo"
3Utilizamos el operador **= para elevar "base" al numero "expo"
4Imprimimos el numero elevado
