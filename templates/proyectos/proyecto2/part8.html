Parte 8

Esta es la parte donde se le va a permitir al usuario escoger la ficha con la que desea jugar, las fichas que se van a manejar son simples caracteres especiales ej. * @ $ &. Esta función es muy similar a la anterior, esta recibe como parámetro dos diccionarios, j1 y j2. y retornará los mismos diccionarios pero esta vez con una clave nueva agregada, que será [“ficha”], que contiene el carácter especial que se haya escogido para cada jugador. Si las fichas escogidas son las mismas entonces retorna dos ceros ya que una partida no puede tener las mismas fichas en juego.

¿Como se realizara?

La primera parte que se va a realizar es crear la lista de fichas disponibles en la cual cada elemento de esta será un string con los caracteres especiales especificados anteriormente, se recorrerá la lista de fichas por índice y se imprime el índice y el elemento correspondiente en ese índice. Luego se pedirá a los usuarios que ingresen cuales son las fichas que desean utilizar. Si las fichas escogidas son diferentes entonces se retorna los diccionarios recibidos por parámetro agregándoles la nueva clave [‘ficha’] y el valor corresponde a la ficha escogida por cada usuario. Si las fichas son iguales entonces retorna dos ceros.

Felicitaciones

Linea # 4

Se debe recorrer la lista de las fichas por índice ¿Cómo se hace esto?

Felicitaciones

Utilizando un for se especifica una variable ficha la cual va a coger un valor entre 0 y la longitud de las lista de fichas, como esta lista es estática la longitud siempre va a ser de 4, por ende la variable ficha coge un valor entre los rango 0 y 4.

Linea #11

Se tiene que retornar los diccionarios actualizados ¿Como se retorna en una función?

Felicitaciones

j1 y j2 son dos diccionarios que contienen la información de dos jugadores registrados, al utilizar j1[‘ficha’] y j2[‘ficha’] lo que se está haciendo es crear una nueva clave llamada ficha para estos dos, usando el return se puede retornar los valores que se le indiquen, en este caso j1 y j2 que ya están actualizados.
