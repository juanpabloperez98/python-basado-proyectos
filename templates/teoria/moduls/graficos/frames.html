1 Frames

Frame() es una función que nos permite crear un contenedor, que como su nombre lo indica contiene o almacena diferentes elementos gráficos que nos proporciona Tkinter. El contenedor debe estar dentro de la raíz para poder ser visualizado, adicional a esto, debemos darle color de fondo, pues por defecto los frames, viene con el color de fondo de la raíz.
A continuación un ejemplo de como crear un frame y poder visualizarlo: 

            
1from tkinter import *
2root = Tk()
3root.config()
4root.geometry("300x300")
5miFrame = Frame()
6miFrame.config(bg="red",width="100",height="100")
7miFrame.pack()
8root.mainloop()
Salida

La función Frame() es la que nos permite crear el contenedor, como podemos ver en el ejemplo, llamamos la función Frame() y la igualamos a una variable llamada miFrame.
Los frames al igual que el root también tienen el método config, que permiten darle características a los componentes a través de atributos y valores, para este ejemplo, se utiliza los atributos bg (para darle color de fondo al frame), width (para definir el ancho) y height(para definir el alto).
Nótese que en la penúltima linea se llama al método pack(), este método es necesario para poder empaquetar el frame dentro de la raíz.

2 Posicionamiento de Frames

Como se ha podido venir viendo, los frames son simples componentes de la librería Tkinter que funcionan como contenedores de diversos elementos, como botones, labels, texto, etc. Como se mostro en la parte de arriba, un contenedor o frame posee un método llamado pack, que sirve para poder empaquetar el frame dentro de la pantalla principal root. Ahora bien, si se sabe que los frames pueden ser empaquetados en la pantalla principal, también estos pueden ser posicionados de acuerdo a unos atributos que se le pasen por parámetro al método pack.
A continuación se muestran los tipos de posicionamientos que se le puede dar a un frame con el método pack:

  • Side
  • Anchor
  • Fill
    • Both
  • Expand 
                    
1from tkinter import *
2root = Tk()
3root.config()
4root.geometry("300x300")
5miFrame = Frame()
6miFrame.config(bg="red",width="100",height="100")
7miFrame.pack(side="right")
8root.mainloop()
Salida

Similar al ejemplo anterior, se crea un contenedor con el método Frame() de la librería Tkinter, anteriormente se utilizaba el método pack() para empaquetar el frame dentro de la pantalla principal root, ahora se realiza de la misma manera, solo que esta vez al método pack, se le pasa como parámetro el atributo side = “Right”, este atributo lo que permite es poder posicionar el frame al lado derecho de la pantalla principal (root).
NOTA: side se puede igualar a los valores left, right, top, bottom. Que equivalen a decir (izquierda,derecha,arriba,abajo) respectivamente. 

                        
    1from tkinter import *
    2root = Tk()
    3root.config()
    4root.geometry("300x300")
    5miFrame = Frame()
    6miFrame.config(bg="red",width="100",height="100")
    7miFrame.pack(side="right",anchor = "n")
    8root.mainloop()
Salida

Anteriormente se mostró el atributo side, se dijo que side, tomaba los valores de left,right,top,bottom para poder colocar el frame en UNA sola posición, que seria izquierda,derecha, arriba y abajo. Ahora con el atributo anchor acompañado del atributo side, se podrá posicionar el frame en DOS posiciones.
El atributo anchor trabaja con las primeras letras de los punto cardenales, north, south, east, west. Es por eso que en el ejemplo anterior se coloca el atributo anchor = “n”, haciendo referencia a que el anchor va a tomar de la posición north (norte en español).
NOTA: usted puede jugar con los atributos del método pack y visualizar los diferentes tipos de posicionamientos que se le puede dar a un frame. 

                    
    1from tkinter import *
    2root = Tk()
    3root.config()
    4root.geometry("300x300")
    5miFrame = Frame()
    6miFrame.config(bg="red",width="100",height="100")
    7miFrame.pack(fill="x")
    8root.mainloop()
Salida

Ya vimos anteriormente cómo se podía posicionar un frame o contenedor en la pantalla principal o raíz del programa. El atributo fill, permite que el frame se extienda extienda a base de la raíz dependiendo el parámetro que se le envíe, para este ejemplo se usa: fill=”x”, lo que permite que cuando se redimensione la pantalla en el eje x, el frame se redimensiona con la raíz. 

                    
    1from tkinter import *
    2root = Tk()
    3root.config()
    4root.geometry("300x300")
    5miFrame = Frame()
    6miFrame.config(bg="red",width="100",height="100")
    7miFrame.pack(fill="y",expand="True")
    8root.mainloop()
Salida

En el ejemplo de fill, se mostró como se expande o redimensiona el frame en el eje x cuando la raíz se expande igual, ahora es por lógica pensar que agregando el fill=”y”, se puede expandir o redimensionar el frame en el eje y. Pues si ya lo intentaron se debieron dar de cuenta que no funciona, y esto es debido a que le falta al método pack, un parámetro adicional diferente al ya puesto fill, que permite decirle al método que se puede expandir en los ejes y. Este atributo se le llama expand y se iguala a un valor booleano. 

                    
    1from tkinter import *
    2root = Tk()
    3root.config()
    4root.geometry("300x300")
    5miFrame = Frame()
    6miFrame.config(bg="red",width="100",height="100")
    7miFrame.pack(fill="both",expand="True")
    8root.mainloop()
Salida

Hasta el momento se ha visto un atributo que permite expandir los frames con respecto a la raíz, como lo es fill. Ahora bien, se mencionó que fill igualandolo a x o y, permite que el frame se expanda con respecto a la raíz en el eje x o y respectivamente, ahora, si se quisiera poder expandir en los dos ejes, se puede utilizar fill=”both”, both lo que indica, es que se pueda expandir tanto en x como en y.
NOTA: el atributo expand, se explica en el siguiente enlace de la lista.

3 Estilos de Frames

Hasta el momento se ha visto cómo crear un frame, los atributos de un frame, y como posicionarlo o expandirlo con respecto a una raíz. Ahora se mostrará ciertos estilos que se le pueden dar a un frame, como darle un borde, o cambiar el estilo del puntero cuando esté por encima del frame.
A continuación se muestran los estilos a explicar:

  • Relief
  • Cursor 
                    
1from tkinter import *
2root = Tk()
3root.config()
4root.geometry("300x300")
5miFrame = Frame()
6miFrame.config(bg="red",width="100",height="100",relief="groove",bd=10)
7miFrame.pack()
8root.mainloop()
Salida

Como se puede observar en la salida, el frame que se creó tiene un color de fondo rojo y un ancho y alto de 100. Nada diferente a lo ya visto, pero adicional a esto se puede observar como se agrega un atributo llamado relief, este atributo lo que permite es colocar el tipo de borde que tiene el frame. Adicional a eso se le agrega el atributo bd=10, esto sirve para agregar el tamaño en píxeles del borde.
NOTA: Los tipos de bordes pueden variar dependiendo el valor que se le de al atributo relief, si desea averiguar más al respecto puede consultar en esta documentación no oficial sobre los estilos de relief 

                    
1from tkinter import *
2root = Tk()
3root.config(bg="blue",bd=10,relief="groove")
4root.geometry("300x300")
5miFrame = Frame()
6miFrame.config(bg="red",width="100",height="100",bd=10,relief="groove",cursor="pirate")
7miFrame.pack()
8root.mainloop()
Salida

En este ejemplo se puede visualizar que a la raíz, también se le puede agregar el atributo relief, como se le ha hecho al frame.Tambien se puede observar que en los parámetros enviados al frame en el método pack, se agrega un nuevo atributo llamado cursor, este atributo sirve para darle estilos al puntero del mouse una vez esté esta encima del frame.
NOTA: los estilos que se le pueden dar al cursor se encuentra en la documentación no oficial.
