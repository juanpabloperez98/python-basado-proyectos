Proyecto 3

Asistente de biblioteca

Este tercer proyecto consiste en crear un software para una biblioteca, la biblioteca necesita un software para poder hacer préstamos de los libros que dispone. Desde el software se debe poder crear nuevos libros, o eliminarlos, consultar si un libro está en préstamo o no, realizar préstamos y devoluciones. En caso de no haber disponibilidad de un libro debe notificarse.

Para tener en cuenta:

  • Al crear un libro nuevo, debe tener en cuenta, la siguiente información: Nombre del libro, autor, año de publicación, copias disponibles, y el ISBN
  • Si se va a crear un nuevo libro se debe validar que ya no esté creado un libro con el mismo ISBN.
  • Al realizar un préstamo se debe tener en cuenta la siguiente información: Cédula de la persona que lo prestó, nombre del libro que prestó, fecha de préstamo y fecha de devolución.
  • Se debe poder listar todos los libros existentes con la información relacionada.
  • En caso de que un usuario intente sacar prestado un libro, para el cual no hay unidades disponibles. Debe notificarle que no es posible prestar el libro
  • El software debe contar con un menú que le muestre al usuario las opciones disponibles.
