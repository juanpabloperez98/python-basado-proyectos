Parte 4

En esta cuarta parte se va a realizar la función de crear un nuevo libro, esta función recibe como parámetro la lista de los libros que hay hasta el momento, esta función se va a encargar de crear un nuevo diccionario con la información del nuevo libro que se desea crear, y adicional a eso agrega el diccionario a la lista libros, esta función retorna la lista de libros actualizada, es decir con el nuevo libro agregado a la lista. También puede devolver la lista de libros sin actualizar, esto cuando el libro que se desea agregar ya se encuentra en la lista, por ende no se actualiza nada.

¿Como se realizara?

Lo primero que se va hacer es declarar una variable llamada libro e igualarla a None, después se pide al usuario que ingrese el isbn del libro que quiere agregar, se hace el llamado a la función buscar_libros(libros, isbn), que si recuerdan es la función del apartado anterior, esta función retorna o un libro o un None si se recuerda, después se valida si libro es igual a None, de ser así es porque no se ha encontrado ningún libro con el isbn ingresado, por ende se puede crear el nuevo libro, luego se pedirán los datos necesario al usuario para crear el libro (título, autor, copias disponibles), se creará el diccionario con estos valores y se agregara a la lista de libros, por último se retorna la lista de libros. Dado el caso que la llamada a la función buscar_libros(libros, isbn) retorne un diccionario, entonces se imprime por pantalla que el libro ya existe y que no se puede crear.
