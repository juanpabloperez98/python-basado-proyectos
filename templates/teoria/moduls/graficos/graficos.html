Gráficos

¿Que son Gráficos ?

Los gráficos, interfaz grafica de usuario ó GUI, es un programa, que actuá como intermediario entre el usuario y el software, utilizando un conjunto de elementos como imágenes y objetos gráficos que permiten representar información y acciones necesarias para poder interactuar con el software de manera fácil y agradable. El principal uso de los gráficos consiste en proporcionar un entorno visual sencillo para permitir la comunicación con el sistema operativo de una maquina o computador.

¿Como vamos a trabajar?

Como se menciono anteriormente las interfaces graficas son un conjunto de elementos que sirven de intermediarios entre los usuarios y el software. En Python existen diversas librerías que nos permiten crear y manejar una interfaz grafica de manera relativamente sencilla. Una de las principales librerías para manejo de gráficos en Python es Tkinter.
Tkinter es una librería grafica de Python, es multiplataforma, lo que permite que se puedan ejecutar programas en diferentes sistemas operativos. Tkinter no es la única librería grafica que existe, hay otras muchas como por ejemplo wxPython , PyQt y kivy. La mayor fortaleza de Tkinter es su facilidad de uso. Funciona de manera inmediata en la mayoría de las plataformas (Linux,OSX,Windows), y complementa con una alta gama de widgets necesarios para las tareas más comunes. (botones, etiquetas, lienzos de dibujo, texto de varias líneas, etc.).
