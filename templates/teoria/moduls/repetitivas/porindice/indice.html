1 Ejemplo de Juan

Véase el ejemplo anterior de juan, se sabe que juan pesa 90 kilo y que cada día montando bicicleta rebaja uno, ¿al finalizar los 10 días del reto cuanto pesara juan?. 

                    
1peso = 90
2dias = 10
3for i in range(dias):
4    peso -= 1
5print("Juan pesa ahora: ",peso)
Salida
Juan pesa ahora: 80

Se especifica en el range, el número de veces que se necesita realizar el ciclo, cada vez que el ciclo se repita el peso de juan se disminuye en 1, de esta manera se puede observar cómo se llega a la respuesta requerida a través de un ciclo for.

2 Imprimir primeros 9 digitos

Que tal que se necesiten los primeros nueve dígitos para hacer un programa que haga las tablas de multiplicar, ¿Cómo se obtendrán estos primeros 9 dígitos? 

                    
1for i in range(10):
2    print(i)
Salida
1 2 3 4 5 6 7 8 9

3 Caracteres de un string

Anteriormente se mencionaron sobre los strings diciendo que son un conjunto de caracteres como números,letras,caracteres especiales (* @ _ ) etc. A continuación se muestra como visualizar cada elemento de un string utilizando un ciclo for por índice. 

                    
1string = "Mundo"
2longitud = len(string)
3for i in range(longitud):
4    letra = string[i]
5    print("Letra # %d  -  %s" % (i,letra))
Salida
Letra # 0 - M
Letra # 1 - u
Letra # 2 - n
Letra # 3 - d
Letra # 4 - o

La palabra reservada len ayuda a obtener la longitud de un elemento iterable como lo es un string (en el siguiente módulo se enseñara sobre otros elementos iterables). De esta manera se obtiene el número de veces que se requiere repetir el ciclo. Los string son elementos iterables lo que significa que para acceder a su contenido se utiliza la expresión [ índice ].

4 Rango específico

Anteriormente se ha visto como la función range coge un rango entre el 0 y el número que se le especifique, es decir que si se envía un 10, se genera un rango entre 0-9, permitiendo repetir el ciclo 10 veces, pero que tal si ahora lo que se necesita es especificar ese rango, que ya no sea entre 0 - 10, sino entre 5-10, para eso se puede especificar en el range, el rango con el que se desea trabajar. 

                    
1for i in range(5,10):
2    print(i)
Salida
5
6
7
8
9

Tan sencillo como en los ejemplos anteriores para definir el rango solo basta con colocar dentro del range los rangos con los que se desea trabajar.
