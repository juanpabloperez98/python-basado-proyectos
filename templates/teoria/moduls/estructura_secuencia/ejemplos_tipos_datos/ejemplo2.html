Ejemplo 2

Enunciado

Declarar una variable de tipo string que contenga un nombre, después se pide que se le concatene a esta variable el string “Bienvenido” y se imprima la variable concatenada

Solución

        
1nombre = 'Juan Andres'
2nombre = nombre + '  Bienvenido'
3print(nombre)
Salida
Juan Andres Bienvenido
Reiniciar

Explicación

        
1Se declara una variable llamada "nombre" e igualamos esta variable al string ‘Juan Adres’
2Igualamos la variable "nombre", a lo que haya en la variable nombre y usando el símbolo +, le concatemos el string ‘Bienvenido’
3Imprimimos lo que haya en la variable "nombre"
