1 Frames

Frame() es una función que nos permite crear un contenedor, que como su nombre lo indica contiene o almacena diferentes elementos gráficos que nos proporciona Tkinter. El contenedor debe estar dentro de la raíz para poder ser visualizado, adicional a esto, debemos darle color de fondo, pues por defecto los frames, viene con el color de fondo de la raíz.
A continuación un ejemplo de como crear un frame y poder visualizarlo: 

            
1from tkinter import *
2root = Tk()
3root.config()
4root.geometry("300x300")
5miFrame = Frame()
6miFrame.config(bg="red",width="100",height="100")
7miFrame.pack()
8root.mainloop()
Salida

La función Frame() es la que nos permite crear el contenedor, como podemos ver en el ejemplo, llamamos la función Frame() y la igualamos a una variable llamada miFrame.
Los frames al igual que el root también tienen el método config, que permiten darle características a los componentes a través de atributos y valores, para este ejemplo, se utiliza los atributos bg (para darle color de fondo al frame), width (para definir el ancho) y height(para definir el alto).
Nótese que en la penúltima linea se llama al método pack(), este método es necesario para poder empaquetar el frame dentro de la raíz.

2 Posicionamiento de Frames

Como se ha podido venir viendo, los frames son simples componentes de la librería Tkinter que funcionan como contenedores de diversos elementos, como botones, labels, texto, etc. Como se mostro en la parte de arriba, un contenedor o frame posee un método llamado pack, que sirve para poder empaquetar el frame dentro de la pantalla principal root. Ahora bien, si se sabe que los frames pueden ser empaquetados en la pantalla principal, también estos pueden ser posicionados de acuerdo a unos atributos que se le pasen por parámetro al método pack.
A continuación se muestran los tipos de posicionamientos que se le puede dar a un frame con el método pack:

  • Side
  • Anchor 
                    
1from tkinter import *
2root = Tk()
3root.config()
4root.geometry("300x300")
5miFrame = Frame()
6miFrame.config(bg="red",width="100",height="100")
7miFrame.pack(side="right")
8root.mainloop()
Salida

Similar al ejemplo anterior, se crea un contenedor con el método Frame() de la librería Tkinter, anteriormente se utilizaba el método pack() para empaquetar el frame dentro de la pantalla principal root, ahora se realiza de la misma manera, solo que esta vez al método pack, se le pasa como parámetro el atributo side = “Right”, este atributo lo que permite es poder posicionar el frame al lado derecho de la pantalla principal (root).
NOTA: side se puede igualar a los valores left, right, top, bottom. Que equivalen a decir (izquierda,derecha,arriba,abajo) respectivamente. 

                        
    1from tkinter import *
    2root = Tk()
    3root.config()
    4root.geometry("300x300")
    5miFrame = Frame()
    6miFrame.config(bg="red",width="100",height="100")
    7miFrame.pack(side="right",anchor = "n")
    8root.mainloop()
Salida

Anteriormente se mostró el atributo side, se dijo que side, tomaba los valores de left,right,top,bottom para poder colocar el frame en UNA sola posición, que seria izquierda,derecha, arriba y abajo. Ahora con el atributo anchor acompañado del atributo side, se podrá posicionar el frame en DOS posiciones.
El atributo anchor trabaja con las primeras letras de los punto cardenales, north, south, east, west. Es por eso que en el ejemplo anterior se coloca el atributo anchor = “n”, haciendo referencia a que el anchor va a tomar de la posición north (norte en español).
NOTA: usted puede jugar con los atributos del método pack y visualizar los diferentes tipos de posicionamientos que se le puede dar a un frame. 

                    
    1from tkinter import *
    2root = Tk()
    3root.config()
    4root.geometry("300x300")
    5miFrame = Frame()
    6miFrame.config(bg="red",width="100",height="100")
    7miFrame.pack(fill="x")
    8root.mainloop()
Salida
