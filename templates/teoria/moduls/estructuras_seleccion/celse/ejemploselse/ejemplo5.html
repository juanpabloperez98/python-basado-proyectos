Ejemplo 5

Enunciado

Pedir al usuario que ingrese por teclado 3 números enteros n1, n2, n3, se desea realizar un programa que imprima por pantalla los números ingresados por el usuario de manera ascendente, es decir, de menor a mayor.

Solución

        
1n1 = int(input("Ingrese el primer número: "))
2n2 = int(input("Ingrese el segundo número: "))
3n3 = int(input("Ingrese el tercer número: "))
4if n1 < n2 and n2 < n3:
5   if n2 < n3:
6       print(n1, n2, n3)
7   else n3 < n2:
8       print(n1, n3, n2)
9elif n2 < n1 and n2 < n3:
10  if n1 < n3:
11      print(n2, n1, n3)
12  else n3 < n1:
13      print(n2, n3, n1)
14elif n3 < n1 and n3 < n2:
15  if n1 < n2:
16      print(n3, n1, n2)
17  else n2 < n1:
18      print(n3, n2, n1)

Explicación

        
1Se pide al usuario que ingrese el primer número por teclado.
2Se pide al usuario que ingrese el segundo número por teclado.
3Se pide al usuario que ingrese el tercer número por teclado.
4Se compara para saber si el primer número es el más pequeño de todos.
5Se compara para saber si el segundo número es menor que el tercero.
6Se imprime en pantalla el primer número, luego el segundo y por último el tercero.
7De no cumplirse la condición anterior, se continua con el resto del programa.
8Se imprime en pantalla el primer número, luego el tercero y por último el segundo.
9De no cumplirse la condición anterior, se compara para saber si el segundo número es el más pequeño de todos.
10Se compara para saber si el primer número es menor que el tercero.
11Se imprime en pantalla el segundo número, luego el primero y por último el tercero.
12De no cumplirse la condición anterior, se continua con el resto del programa.
13Se imprime en pantalla el segundo número, luego el tercero y por último el primero.
14De no cumplirse las dos condiciones anteriores, se compara para saber si el tercer número es el más pequeño de todos.
15Se compara para saber si el primer número es menor que el segundo.
16Se imprime en pantalla el tercer número, luego el primero y por último el segundo.
