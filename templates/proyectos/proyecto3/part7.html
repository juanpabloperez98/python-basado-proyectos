Parte 7

En esta penúltima parte se la a realizar la función que permitirá ver los libros ingresados hasta el momento, esta función se va a llamar ver_libros y recibirá como parámetro la lista de libros, esta función a diferencia de las anteriores no retornara nada, solo imprimirá la información que contiene la lista de libros.

¿Como se realizara?

Se declara la función que recibe como parámetros la lista de libros, luego a través de un ciclo for por elementos iterables se obtendrán los diccionarios que contienen cada una de la información de los libros guardados hasta el momento, luego se imprimirá la información de cada uno de los libros, como por ejemplo el isbn, título, autor y copias disponibles.

Vamos a realizarlo

Felicitaciones
Puedes pasar a la siguiente parte

Solución

              
      1def ver_libros(libros):
      2    print("***Libros disponibles***")
      3    for i in libros:
      4        print("isbn",i['isbn'])
      5        print("titulo",i['titulo'])
      6        print("autor",i['autor'])
      7        print("copias disponibles",i['copias'])

Linea # 1

Se debe declarar la función ver_libros, ésta recibe como parámetro la lista libros ¿Cómo se haría esto?

Felicitaciones

Utilizando la palabra reservada def se declara la función con el nombre y se le pasan los parámetros, en este caso solo uno que es la lista libros.

Linea # 3

Se necesita recorrer la lista de libros a través de un ciclo for por elementos iterables ¿Cómo hacer esto?

Felicitaciones

La lista de libros contiene como elementos un conjunto de diccionarios donde cada diccionario contiene la información de un libro específico, en cada iteración, la variable i, cogerá como valor esos diccionarios.

Linea # 7

Se requiere imprimir el número de copias disponibles que hay del libro obtenido en la variable i ¿Como se hace esto?

Felicitaciones

En la variable i hay almacenado un diccionario, para acceder al valor de una clave del diccionario, solo basta con colocar entre los corchetes el nombre de la clave, en esta caso ‘copias’
