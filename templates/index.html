Enseñanza del lenguaje python basado en proyectos

Enseñanza del lenguaje python basado en proyectos, es un software creado por la Universidad Tecnológica de Pereira, que tiene como objetivo explicar al alumno sobre este maravilloso e increíble lenguaje de programación. En este software se pueden señalar dos partes importantes. La parte teórica y la parte practica, en la parte teórica se tiene como objetivo poder enseñar los aspectos mas importantes con respecto a este lenguaje de programación, su estructura, su sintaxis y su manejo a la hora de llevar un problema planteado. La parte practica busca generar en el alumno la capacidad de poder sobrellevar diferentes problemas que se pueden presentar a la hora de realizar un proyecto.

¿Que vamos aprender?

Como se menciono anteriormente el software maneja dos partes importantes, que son la parte teórica y la parte practica. En la parte teórica aprenderemos sobre los aspectos mas relevantes de este lenguaje de programación. Aprenderemos todo lo relacionado con las estructuras de secuencia, tal como, declarar una variable, ver los tipos de variables, como se manejan los operadores aritméticos, los operadores lógicos, etc. También se aprenderá sobre todo lo relacionado con estructuras de selección y estructuras repetitivas, los ciclos, y las condicionales. A su vez se manejaran las estructuras de datos como listas, matrices, y cadenas. Por ultimo se explicara todo lo relacionado con funciones,recursividad y archivos. Adicional a eso se manejara una sección donde se explicara el manejo de gráficos con una librería conocida como Tkinter.
En la parte practica, se busca desarrollar en el estudiante la capacidad de poder resolver problemas de programación en el lenguaje python. La metodología con la que se busca enseñar todo lo aprendido en la sección de teoría, es basándonos en proyectos. Con lo cual, la parte practica cuenta con 6 proyectos, los cuales cuentan con una manera didáctica de resolverlos. Se le recomienda al estudiante que mientras se va desarrollando el proyecto en el software, este lo vaya realizando en su propio computador. Al final de cada proyecto se le entregara un link al estudiante en donde podrá encontrar, el repositorio en donde esta alojado cada proyecto.

Lenguaje Python

Antes de empezar con todo, es bueno aprender un poco sobre lo que es el lenguaje python, sus ventajas, sus desventajas y cuales son sus principales características las cuales lo hacen uno de los lenguajes mas llamativos a la hora de aprender programación.
Python es un lenguaje de programación de propósito general muy poderoso y flexible, a la vez que sencillo y fácil de aprender. Es un lenguaje de alto nivel, que permite procesar fácilmente todo tipo de estructuras de datos, tanto numéricos como de texto.
Tiene una gran variedad bibliotecas, las cuales permiten el desarrollo de casi todo tipo de software, como lo son aplicativos webs, GUIs de escritorio, programación de redes, desarrollo de software y juegos, entre otras.
Este lenguaje de programación tiene características significativas las cuales lo hacen diferente a muchos otros lenguajes de programación existentes. Por ejemplo: Python es un lenguaje interpretado, es decir, no es compilado, a diferencia de otros lenguajes como lo son C ó C++, es multiplataforma, lo cual nos permite poder ejecutar su código en cualquier sistema operativo, es un lenguaje multiparadigma, el cual soporta varios paradigmas de programación, etc.
