1 Label

Label() es un método que maneja la librería Tkinter, y sirve para crear etiquetas de texto o imágenes, ¿esto qué quiere decir?, significa que se puede crear contenido de texto o contenido de imágenes que se podrán implementar dentro de un contenedor frame. 

            
1from tkinter import *
2root = Tk()
3miFrame = Frame()
4miFrame.config(bg="red",width="200",height="200")
5miFrame.pack()
6miLabel = Label(miFrame, text = "Este es mi label")
7miLabel.pack()
8root.mainloop()
Salida

Como se puede ver, se utiliza el método Label para crear la etiqueta y se iguala a la variable miLabel. La sintaxis del método Label es sencilla, el primer parámetro que recibe es el contenedor donde será almacenado esta etiqueta o label, y el resto de parámetros, vendrán siendo una serie de opciones que permite la etiqueta label. Las opciones que recibe esta etiqueta son muchas, para este ejemplo utilizamos el parámetro text, que permite indicarle el texto que va a llevar esta etiqueta.
NOTA: nótese que se especificó el frame con un ancho, un alto y un color de fondo correspondiente, pero no se está respetando el tamaño asignado ni tampoco el color de fondo, esto es debido a que al crear la etiqueta o label, utilizamos el método pack para empaquetarla dentro del frame especificado. Esto hace que el frame tome el tamaño del label y el color de este mismo.

2 Label (place)

Como ya se vio, Label() es el método que nos permite crear la etiqueta de texto o imagen para poder incluirla dentro de un contenedor o frame. Pero adicional a eso se vio que el método pack, empaqueta el label dentro del frame sin respetar los atributos de configuración establecidos al frame. Para ello existe otro método que permite posicionar los labels dentro del frame, sin omitir o ignorar los atributos de configuración del frame. A continuación se muestra un ejemplo: 

            
1from tkinter import *
2root = Tk()
3miFrame = Frame()
4miFrame.config(bg="red",width="200",height="200")
5miFrame.pack()
6miLabel = Label(miFrame, text = "Este es mi label")
7miLabel.place(x=50,y=50)
8root.mainloop()
Salida

El método place permite ubicar el label dentro del frame según la posición que se le indique, a este método se le pasan dos parámetros (x,y), estos parámetros sirven para indicar en qué parte del frame se desea ubicar el label. Los parámetros (x,y) se representan en pixeles. NOTA: si a los parámetros (x,y) se le envían valores mayores a las dimensiones que tiene el frame contenedor, el label no se podrá visualizar pues está excediendo las dimensiones del frame que lo contiene.
