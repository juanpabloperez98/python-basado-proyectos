1 Tk()

Tk() es una función de la librería Tkinter, la cual nos representa la raíz o la iniciación de toda la interfaz grafica que se va a manejar, en otras palabras, esta función nos devolverá la pantalla principal en la cual podremos incluir algunas configuraciones e implementar algunos elementos de los mencionados anteriormente. A continuación un pequeño ejemplo: 

                
1from tkinter import *
2root = Tk()
3root.mainloop()
Salida

Como se pudo ver, se llamo a la función Tk() y la igualamos a la variable root (El nombre de la variable es un estándar, mas sin embargo no es obligatorio que se llame así, puede tener el nombre que uno desee), Después utilizamos la función mainloop() que es la que nos permite que la ventana este en ejecución y de esta manera no se cierre.

2 Atributos del Tk()

Ya vimos como crear nuestra primera ventana con la función Tk(), ahora veremos como darle ciertas características a la ventana principal, asignándole una serie de atributos que permitirán personalizar un poco esta ventana root. A continuación se muestra los tipos de atributos que se pueden manejar para personalizar nuestra ventana principal:

  • Agregar titulo
  • Permitir redimensionar
  • Agregar tamaño a la ventana
  • Agregar color de fondo 
                        
1from tkinter import *
2root = Tk()
3root.title("Mi ventana")
4root.mainloop()
Salida

En el ejemplo podemos visualizar como aparece en la parte superior de la ventana el nombre que le pusimos con la función title 

                        
1from tkinter import *
2root = Tk()
3root.resizable(True,False)
4root.mainloop()
Salida

La función resizable, es la que nos permite redimensionar la pantalla y en que ejes poder redimensionar. Esta función recibe como parámetro dos valores booleanos True ó False. El primer parámetro que reciba es el que hace referencia a las ejes X y el segundo parámetro que reciba es el que hace referencia a las ejes Y. Si el parámetro es True, entonces nos permite redimensionar en ese eje, si el parámetro por el contrario es False, no nos permitirá redimensionar en ese eje.
NOTA: La manera de redimensionar es con el mouse, acercar el mouse a los bordes de las pantallas y ver como le permiten redimensionar el root 

                        
1from tkinter import *
2root = Tk()
3root.geometry("500x500")
4root.mainloop()
Salida

La función función geometry, es la encargada de permitirnos poder crear la ventana principal o el root, con las dimensiones que nosotros queramos.
NOTA: es importante tener en cuenta que las dimensiones deben ir como un string (entre comillas), de lo contrario nos mostrara un error. 

                        
1from tkinter import *
2root = Tk()
3root.geometry("300x300")
4root.config(bg="blue")
5root.mainloop()
Salida

config es una método que nos permite agregar muchos atributos a un elemento de Tkinter, los atributos que se pueden agregar varían dependiendo del elemento a tratar. Si se quieren mirar los atributos que posee el elemento solo basta con imprimir:

1print(root.configure().keys())

El atributo que nos interesa a nosotros para este ejemplo es el bg, que es el atributo que hace referencia al color de fondo, o en ingles Background. Este atributo nos colocara el fondo del color que le especifiquemos, en este caso es el color azul. Para conocer los distintos colores que maneja tkinter recomendamos leer aquí.

NOTA: si se quiere tomar mas información acerca de los elementos y atributos de tkinter, recomendamos ver la documentación oficial de la librería.
