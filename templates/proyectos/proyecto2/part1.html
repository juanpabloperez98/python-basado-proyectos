Parte 1

Esta es la primera parte del segundo proyecto, similar que el proyecto anterior se va a crear la función main, en la cual se va hacer el llamado a la función menu() y a la función llamada_funciones(). La primera función mostrará el menú y la segunda es la función que va a realizar el llamado a las demás funciones.

¿Como se realizara?

Primero se va a crear la función main, después se declara una lista y se llamará jugadores, esta lista guardará los jugadores que se vayan creando. Luego se va a crear un ciclo while que va hacer que el programa no se cierre hasta que se le indique, para eso primero se va a declarar una variable llamada fin y se igualara a un booleano False, después se declara el ciclo while diciendo que se ejecute mientras fin sea igual a False. Dentro del ciclo while se va hacer el llamado a la función menu (función que se realizará en el apartado 2), luego se pedirá al usuario que ingrese una opción y se valida si opción es diferente de 4, de ser así se hará el llamado a la función llamada_funciones y se le pasará como parámetro la opción escogida por el usuario y la lista vacía jugadores
