Estructuras de datos

¿Que son las estructuras de datos?

Hablar de estructuras de datos es hablar de una forma particular de cómo organizar los datos que se manejan en una computadora para que estos puedan ser utilizados de manera más rápida y eficiente. Las estructuras de datos no son más que un medio para manejar grandes cantidades de información de una manera fácil y rápida. A medida que se ha ido avanzando en esta guía de programación de python, en la parte de la teoría se ha podido evidenciar o se ha hablado de lo que son las cadenas de caracteres o listas. Pues bien, estas son consideradas estructuras de datos, que permiten agrupar diversos tipos de datos dentro de ellas, para así poder manejar de manera ordenada y óptima la información que se presente, en este módulo se hablarán de las estructuras de datos mas comunes en python.
