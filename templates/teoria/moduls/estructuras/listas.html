4.2.1¿Qué son las listas?

Las listas al igual que las cadenas de caracteres son una estructura de datos, que tiene como objetivo guardar y listar un conjunto de elementos heterogéneos, es decir un conjunto de elementos que no necesariamente deben ser iguales. En apartados anteriores se ha mencionado algo acerca de las listas, y como se ha podido venir viendo, las listas contienen tipos de datos que después podrán ser obtenidos de manera similar a como se hacen con las cadenas de caracteres. Por lo general en la mayoría de lenguajes de programación las listas deben ser homogéneas, pero en python esto no importa, pues el lenguaje permite tener listas con elementos heterogéneos. Las listas en Python son variables que almacenan arrays, internamente cada posición puede ser un tipo de datos distinto.

4.2.2 Sintaxis y manejo de listas

La sintaxis para declarar una lista es similar a como declarar una variable de cualquier tipo de dato, se debe colocar el nombre de la variable y luego igualar a dos corchetes vacíos (si se quiere crear una lista vacía) o los elementos de la lista dentro de los corchetes.

4.2.3 Métodos con listas

Las listas tienen unos métodos los cuales permiten trabajar con ellas de manera sencilla y sin tanto conflicto, por ejemplo imagínese que se desea realizar un programa que lleve el registro de los usuario que han utilizado el software, para guardar la información dentro de una lista es necesario hacer una operación de agregar elementos a la lista, o tal vez se requiera eliminar un registro, entonces se debe realizar la operación de eliminar de la lista, hay muchas tareas que se pueden realizar con una lista, y para eso existen unos métodos que permiten trabajar con estas.
