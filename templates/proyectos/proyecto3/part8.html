Parte 8

Ya en esta parte final, se realizará la función de ver los préstamos que se han realizado, es similar a la función de ver_libros(), esta función va a recibir como parámetro la lista de préstamos e imprimirá la información correspondiente a cada uno de los diccionarios que contiene esta lista.

¿Como se realizara?

Lo primero que se va a realizar es crear la función ver_prestamos, que recibirá como parámetro la lista de préstamos, luego se recorre la lista a través de un ciclo for por elementos iterables, y por último se imprimirá la información correspondiente de cada préstamo, como por ejemplo, nombre, cédula, fecha en que lo prestó, fecha en la que lo devuelve y el nombre del libro.

Linea # 1

Se debe declarar la función ver_prestamos, ésta recibe como parámetro la lista prestamos ¿Cómo se haría esto?

Felicitaciones

Utilizando la palabra reservada def se declara la función con el nombre y se le pasan los parámetros, en este caso solo uno que es la lista prestamos.

Linea # 3

Se necesita recorrer la lista de prestamos a través de un ciclo for por elementos iterables ¿Cómo hacer esto?

Felicitaciones

La lista de prestamos contiene como elementos un conjunto de diccionarios donde cada diccionario contiene la información de un prestamo específico, en cada iteración, la variable i, cogerá como valor esos diccionarios.

Linea # 8

Se requiere imprimir el titulo del libro que se presto ¿Como se hace esto?

Felicitaciones

Como se ha explicado, i coge el valor del diccionario correspondiente que contiene la información del préstamo, cuando se creó la función de crear_prestamos(), el diccionario del préstamo que se creaba contenía al diccionario del libro correspondiente, por lo que se crea un diccionario que dentro tiene como valor otro diccionario, de esta manera se accede a la clave [‘libro’] que es el diccionario del libro y luego a ese diccionario del libro se accede a la clave [‘titulo’]
