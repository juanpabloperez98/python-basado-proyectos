6.1.1 Tkinter

Como se menciono en la sección de “Instalar Tkinter”, esta librería viene por defecto en las versiones de Python 3 o mayores, lo que significa que para poder utilizarla en nuestro programa o entorno de trabajo solo basta con importarla de esta manera: 

                  from tkinter import *

NOTA Si se presenta algún problema o la librería no esta instalada por defecto, le recomendamos que revise la sección “Instalar Tkinter” que se encuentra en la barra de navegación del modulo Gráficos.

Tkinter maneja muchos componentes gráficos que permiten realizar toda una interfaz grafica profesional y de manera agradable, algunos de estos componente son:

  • root / raiz
  • Frames
  • Labels
  • Entrys

6.1.2 Root / Raíz

Se le conoce como root o raíz, ya que es el componente principal donde se inicializa toda la interfaz grafica, esta definido por la llamada a la función Tk() de la librería Tkinter. Es un estandar llamar la variable root o raiz mas sin embargo no es determinante a la hora de crear un programa, a continuación se muestra como realizar una interfaz grafica con la función Tk():

6.1.3 Frames

Los frames o marcos, son una especie de contenedor, el cual nos permite al igual que el root o raíz, poner elementos dentro de el. En otras palabras un frame, es como una raíz, pero a diferencia de esta, el frame no se representa como una venta propiamente, sino mas bien como un elemento que contiene mas elementos y que se puede ubicar dentro de la ventana principal o raíz.

  • Interfaz grafica con frames

6.1.4 Labels

Los labels son conocidos como wigets o componentes para poder mostrar texto o imágenes. Tienen como finalidad mostrar un tipo de contenido(texto o imagen), el cual no se podrá interactuar con él, es decir no se puede ni arrastrar, ni borrar, etc. Si se recuerda en el apartado de los frames, se decía que un frame es un tipo de contenedor que permite valga la redundancia contener diferentes tipos de elementos, bueno los labels, son uno de esos tipos de elementos que pueden estar contenidos dentro de un frame.
