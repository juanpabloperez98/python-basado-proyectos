1 Sumar varias veces

Un estudiante de bachillerato quiere sumar varias veces diferentes números, la idea es que él se demore el menor tiempo posible para acabar rápido. 

                    
1def suma(a,b):
2   c = a + b
3   return c
4print(suma(1,2))
5print(suma(3,4))
6print(suma(2,3))
Salida
3 7 5

Al crear la función definida por el estudiante, solo basta con ingresarle los parámetros a y b los cuales son necesarios para que se ejecute el código y de como resultado la suma de estos dos, la suma es almacenada en la variable c la cual posteriormente es retornada con el resultado.

2   if a%2 == 0:

4   else:

Basta con llamar la función con cualquier número dentro y el programa imprime true o false.
