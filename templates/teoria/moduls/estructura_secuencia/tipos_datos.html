1.1.1 Variables

Una variable es un nombre que se refiera a un objeto que reside en memoria. El objeto por lo general es de un tipo de dato especifico ya sea numero o cadena de texto o alguno de los otros tipos existentes en Python. Cada variable debe tener un nombre único llamado identificador. Eso es muy de ayuda pensar las variables como contenedores que contienen data el cual puede ser cambiado después a través de técnicas de programación.

1.1.2 Alcance de las variables

Las variables en Python son locales por defecto. Esto quiere decir que las variables definidas y utilizadas en el bloque de código de una función, sólo tienen existencia dentro de la misma, y no interfieren con otras variables del resto del código. A su vez, las variables existentes fuera de una función, no son visibles dentro de la misma.En caso de que sea conveniente o necesario, una variable local puede convertirse en una variable global declarándola explícitamente como tal con la sentencia global.

1.1.3 Ejemplos de variables

A continuación se presentan unos ejemplos de variables. Se creará un par de variable a modo de ejemplo. Uno de tipo cadenas de caracteres ó string y otro de tipo entero, mas adelante se explicaran sobre los tipos de datos existentes 

                  c = "Hola mundo"
e = 21

Como nos pudimos dar cuenta, en Python no es necesario declarar el tipo de variable que se desea manejar, a diferencia de muchos otros lenguajes.

1.1.4 Ejemplo cambiar valor a una variable

A continuación, se cambiará el valor para una variable de tipo cadenas de caracteres a modo de ejemplo: 

                  c = "Hola soy una variable"
c = "Hola soy la variable cambiada"

Como pudimos ver, primero se le asigno a la variable c el valor de una cadena de caracteres, y posteriormente volvimos a igualar la misma variable c con una nueva cadena de caracteres, de esta manera se esta actualizando el valor de la variable c de manera sencilla.

1.1.5 Ejemplo asignar múltiples valores

A continuación se crearan múltiples variables y asignandoseles múltiples valores: 

                  a,b,c = 5,3.2,"Hola Mundo"

Si se desea asignar el mismo valor a múltiples variables al mismo tiempo, se puede hacer lo siguiente: 

                  x = y = z = True

1.1.6 Palabras reservadas

Existen ciertas palabras que tienen significado especial para el intérprete de Python. Estas no pueden utilizarse para ningún otro fin (como ser nombrar valores) excepto para el que han sido creadas. Algunas de estas son:

  • in
  • is
  • break
  • print
  • not
  • del
  • import

1.1.7 Tipos de datos en python

Como se especifico anteriormente, para declarar variables en Python no es necesario declarar el tipo de dato que puede almacenar.
El funcionamiento de los programas Python depende de los datos que maneja. Todos los valores de datos en Python son objetos, y cada objeto, o valor, tiene "un tipo". Tener en cuenta que, cada tipo de objeto determina qué operaciones va a soportar el objeto y, por lo tanto, qué operaciones se van a poder realizar con esos valores de los datos, qué atributos tiene y si va a poder ser "mutable" o no.
