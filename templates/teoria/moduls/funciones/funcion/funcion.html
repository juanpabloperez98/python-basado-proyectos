5.1.1 ¿Qué es una función?

Una función es un bloque de código con un nombre asociado, que recibe cero o más argumentos como entrada, sigue una secuencia de sentencias, la cuales ejecuta una operación deseada y devuelve un valor y/o realiza una tarea, este bloque puede ser llamados cuando se necesite. La sentencia def es una definición de función usada para crear objetos funciones definidas por el usuario. Una definición de función es una sentencia ejecutable. La definición de función no ejecuta el cuerpo de la función; esto es ejecutado solamente cuando la función es llamada. 

                
Una persona desea crear una calculadora con la que
pretende hacer las 4 operaciones básicas como suma,
resta, multiplicación y división. Para lograr esto
debe hacer un menú en el que el usuario elija cual 
es la operación que desea realizar. Pero cada que 
el usuario entra a alguna opción, la persona observa
que debe poner el menú al final. Aquella persona
quiere evitar escribir tantas veces las mismas lineas
de código para hacer el menú.¿Cómo lo hace?

La solución a este problema es simple, solo debe crear una función definida por el mismo, la cual se va a llamar menú y en ella se va a imprimir el menú. Para mostrar el menú cada que el usuario queira volver a elegir otra opción, basta con llamar la función "menú" al final de cada opción y de esta manera, la persona se evita tener que copiar tantas lineas de cógido.

5.1.2 Sintaxis

Teniendo un poco de idea de lo que es una función, a continuación se explica sobre su sintaxis. Python maneja la sintaxis para definir una función de una manera muy simple, pues para definirla solo basta con poner el comando def, su nombre y los parámetros de entrada y salida que se quieren, a continuación se muestra cómo:


Su estructura básica es:

                
    def nombre_de_la_funcion(Parámetros de entrada):

        Bloque de código ...

        return [Parámetros de salida]

Para que la función se comporte correctamente en el programa, hay que entender que en primer lugar el nombre_de_la_funcion es el mismo con el cual se va a llamar la función en el resto del desarrollo del programa, en segundo lugar, los parámetros de entrada son los parámetros necesarios para que la función se ejecute correctamente y con los cuales pueda cumplir su objetivo. Por último, el comando return es quien va a retornar los parámetros que resultan después de ejecutar el bloque de código que hay dentro de la función.
