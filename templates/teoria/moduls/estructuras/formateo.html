1 Formateo %

El carácter modulo % es un operador integrado en Python. Ese es conocido como el operador de interpolación. Usted necesitará proveer el % seguido por el tipo que necesita ser formateado o convertido. El operador % entonces substituye la frase ‘%tipodato’ con cero o mas elementos del tipo de datos especificado: 

                    
1raiz = 2 ** 2
2imprimir = "La raiz cuadrada de %d es %d"
3print(imprimir % (2,raiz))
Salida
La raiz cuadrada de 2 es 4

2 Sintaxis para el formateo %

El formateo % presenta una sintaxis simple, en el caso anterior se utilizó el %d determinando que el formateo de la cadena iba a contener números decimales, es decir que hay que especificar el tipo de dato con el que sea desea formatear, a continuación los tipos de datos con los que se pueden concatenar:

  • %c = str, simple carácter.
  • %s = str, cadena de carácter.
  • %d = int, enteros.
  • %f = float, coma flotante.
  • %o = octal.
  • %x = hexadecimal.

3 format()

Este método devuelve una versión formateada de una cadena de caracteres. Las substituciones son identificadas entre llaves { } dentro de la cadena de caracteres (llamados campos de formato), y son sustituidos en el orden con que aparecen como argumentos de format(), contando a partir de cero (argumentos posicionales). 

                    
1edad = 20
2nombre = "Juan"
3print("El nombre es {} y la edad es {}".format(nombre,edad))
Salida
El nombre es Juan y la edad es 20
