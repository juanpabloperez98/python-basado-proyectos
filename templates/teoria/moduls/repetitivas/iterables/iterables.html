1 Ejemplo del String

En la parte de ciclos por índice, se puso el ejemplo de cómo recorrer un string con un ciclo for por índice, ahora se va a realizar el mismo ejemplo pero esta vez utilizando el ciclo for por elementos iterables. 

                    
1string  = "Mundo"
2for i in string:
3    print("Letra: ",i)
Salida
Letra: M
Letra: u
Letra: n
Letra: d
Letra: o

Sí recuerda, es el mismo ejemplo, pero esta vez no se necesitó utilizar la longitud del string para manejar el ciclo for, al utilizar la palabra reservada in sin la función range, python itera sobre el elemento iterable obteniendo cada ítem o componente de este, es decir que por cada iteración que python haga, este guardará en la variable i la letra correspondiente a la iteración (M u n d o).

2 Iterando una lista

Una lista simplemente es una estructura que permite guardar diferentes elementos en ella. Como un tren, en el que cada uno de sus vagones permiten llevar pasajeros, así podría ser vista una lista. Las listas son elementos iterables ya que en ella guardan diversos tipos de datos (strings, enteros, flotantes, etc), para iterar sobre una lista se puede hacer de la misma manera que con un string. 

                    
1lista = [1,2,3,4,5]
2for i in lista:
3    print(i)
Salida
1 2 3 4 5

De la misma manera que con un string, la lista coge cada elemento de esta y lo guarda en la variable i, de esta manera cada vez que el ciclo se repita, el valor de i corresponderá al elemento de la lista correspondiente, es decir que i toma el valor de [1,2,3,4,5].

3 Suma los elementos

Si se tiene una lista con ciertos números, y se quiere mostrar la suma de todos ellos se tendrá que realizar algo parecido a esto: 

                    
1numeros = [2,6,7,8]
2x = 0
3for i in numeros:
4    x+=i
5print("Suma de los numeros: ",x)
Salida
Suma de los numeros: 23

Se recorren cada uno de los elementos de la lista y la variable x que empieza en 0, se irá aumentando en el valor que coja i.

4 Suma entre dos elementos

Anteriormente se ha visto como la función range coge un rango entre el 0 y el número que se le especifique, es decir que si se envía un 10, se genera un rango entre 0-9, permitiendo repetir el ciclo 10 veces, pero que tal si ahora lo que se necesita es especificar ese rango, que ya no sea entre 0 - 10, sino entre 5-10, para eso se puede especificar en el range, el rango con el que se desea trabajar. 

                    
1numeros = [1,2,3,4,5]
2numeros2 = [6,7,8,9,10]
3for x,y in zip(numeros,numeros2):
6    suma = x + y
7    print(x,y,' = ',suma)
Salida
1 6 = 7
2 7 = 9
3 8 = 11
4 9 = 13
5 10 = 15

La palabra reservada zip, sirve para agrupar o comprimir dos elementos iterables, lo que quiere decir que cada vez que se recorra un ciclo los valores de las listas numeros y numeros2 serán almacenados en las variables x,y respectivamente. Es necesario que las listas tengan el mismo tamaño o longitud.
