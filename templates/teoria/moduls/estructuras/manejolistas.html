1 Declarar una lista

Para declarar una lista solo basta con colocar el nombre de la variable e igualarla a los corchetes vacíos(si lo que se quiere es crear una lista vacía), o con los elementos de la lista dentro de los corchetes si lo que se requiere es crear la lista con elementos 

                    
1lista_vacia = []
2lista_numeros = [1,2,3,4,5,6,7]
3print("Lista vacia",lista_vacia)
4print("Lista numeros",lista_numeros)
Salida
Lista vacias []
Lista numeros [1, 2, 3, 4, 5, 6, 7]

2 Acceder a un elemento de la lista

En las listas al igual que en las cadenas de caracteres, se puede acceder a sus elementos utilizando los corchetes y especificando el índice en el que se encuentra el elemento, hay que recordar que el índice del primer elemento de la lista es 0. 

                    
1lista = ["Esto es una cadena",2,5.6,"Esto es otra cadena"]
2elemento = lista[0]
3print(elemento)
Salida
Esto es una cadena

Se tiene una lista con elementos de diferentes tipos de datos, al acceder al indice 0, se esta cogiendo el primer elemento de la lista, que en este caso sería la cadena “Esto es una cadena”

3 Recorrer una lista

Recorrer una lista similar a recorrer una cadena de caracteres, la diferencia, es que por cada iteración que se haga en el ciclo for, esta vez no se cogerá un carácter, sino que se cogerá un elemento de la lista, lo que quiere decir que si se tiene una lista con un entero, una cadena de caracteres, y un float, por cada iteración del ciclo for, esté cogerá el entero, la cadena de caracteres y el float respectivamente. 

                    
1lista = [15,"Hola mundo",9.5]
2for i in lista:print(i)
Salida
15
Hola mundo
9.5

Como se puede observar, en cada iteración que se hace con el ciclo for, la variable i va a obtener el valor de cada uno de los elementos, en este caso 15, Hola mundo y 9.5

4 Listas anidadas

Ya se tiene entendido lo que es una lista, se resume como una estructura que permite guardar y listar tipos de datos de manera secuencial, pero ahora suponga que se necesita almacenar en una lista los datos personales de un cliente, para este tipo de cosas se usan las listas anidadas. 

                    
1datos = [
2    ["Juan Pablo",21,10992345],
3    ["Andres David",19,102349924],
4    ["Carlos Jose",25,1293999412],
5]
6for i in datos:
7    print(i)
Salida
['Juan Pablo', 21, 10992345]
['Andres David', 19, 102349924]
['Carlos Jose', 25, 1293999412]

Antes se manejaban las listas con elementos tales como enteros, cadenas de caracteres, etc. Ahora se manejan listas dentro de una lista, esta forma de hacerlo sirve mucho para manejar matrices, o también tener de manera organizada la información.

5 Recorrer listas anidadas

Similar a como se ha venido haciendo, para recorrer una lista anidada, ya no solo basta con realizar un ciclo for, pues al hacer esto, se estaría obteniendo otra lista, ahora es necesario hacer dos ciclos for, de esta manera. 

                    
1datos = [["Juan Pablo",21,10992345]]
2for i in datos:
3    for p in i:print(p)
Salida
Juan Pablo
21
10992345

Para este ejemplo se va a manejar una sola lista dentro de la lista datos, el primer ciclo for, representa el recorrido de la lista datos, es decir, va a coger por cada iteración (en este caso solo una), otra lista, que contiene los datos de un usuario (Nombre,edad,documento), el segundo ciclo for que se realiza es para poder obtener los datos de esa lista que se obtuvo.
